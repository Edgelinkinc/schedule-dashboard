Schedule data syncs automatically to SharePoint via Power Automate. No setup needed — upload a schedule and it will be available on all devices instantly.
How it works:
1. Upload a schedule file → data saves to SharePoint automatically
2. Any device opening the dashboard loads the latest data from SharePoint
3. Personnel changes also sync across all devices

Flows used:
• Get Schedule Data — reads from SharePoint on load
• Save Schedule Data — writes to SharePoint on upload
